Bienvenue sur Midi Piano Web Visual

Piano Visuals vous permet de jouer et de visualiser vos fichiers MIDI directement dans le navigateur. Choisissez un thème, un instrument, importez un fichier .mid ou utilisez nos démos, puis laissez tomber les barres sur le clavier !

À propos de ce projet

Développé en React et Tone.js, ce site supporte le tactile, le clavier d’ordinateur et les contrôleurs MIDI USB pour une expérience réaliste de piano virtuel. Idéal pour l’apprentissage, la pratique ou le fun !
